inet (Internet Addressing)
EtcPal » C++ Wrappers

Overview

C++ utilities for the inet (Internet Addressing) module.

Provides C++ wrappers for IP addresses (IpAddr), socket addresses (SockAddr) and MAC addresses (MacAddr).
