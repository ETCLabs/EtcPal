lwpa » lwpa_timer

Overview

A millisecond-resolution timer.

The times are represented in milliseconds by a 32-bit unsigned integer. Since the timer is monotonically-increasing, wraparound is handled by doing comparisons of the form

timeA - timeB > 0

rather than

timeA > timeB

.
